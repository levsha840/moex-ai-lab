# 06_SESSION_START — MOEX AI LAB

## Как начать следующую сессию

1. Открыть `D:\MOEX_AI` в VS Code.
2. Активировать окружение:

```powershell
.\.venv\Scripts\activate
```

3. Проверить Python:

```powershell
python --version
```

Ожидается Python 3.12.x.

4. Запустить тесты:

```powershell
python -m pytest
```

5. Проверить Git:

```powershell
git status
```
